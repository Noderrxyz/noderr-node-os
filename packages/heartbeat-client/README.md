# @noderr/heartbeat-client

Heartbeat client for Noderr nodes - sends periodic heartbeats to auth-API

## Installation

```bash
pnpm add @noderr/heartbeat-client
```

## Usage

```typescript
import { /* exports */ } from '@noderr/heartbeat-client';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
