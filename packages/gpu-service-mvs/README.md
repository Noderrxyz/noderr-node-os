# @noderr/gpu-service-mvs

Simplified GPU detection service for Noderr Protocol (Linux only)

## Installation

```bash
pnpm add @noderr/gpu-service-mvs
```

## Usage

```typescript
import { /* exports */ } from '@noderr/gpu-service-mvs';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
