# @noderr/node-onboarding

Complete node onboarding flow for all Noderr node tiers

## Installation

```bash
pnpm add @noderr/node-onboarding
```

## Usage

```typescript
import { /* exports */ } from '@noderr/node-onboarding';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
