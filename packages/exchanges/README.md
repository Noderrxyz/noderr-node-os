# @noderr/exchanges

Exchange connectivity service with paper trading engine for testnet simulation

## Installation

```bash
pnpm add @noderr/exchanges
```

## Usage

```typescript
import { /* exports */ } from '@noderr/exchanges';
```

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
