# @noderr/on-chain-settlement

On-chain settlement layer with consensus validation for Noderr Node OS

## Installation

```bash
pnpm add @noderr/on-chain-settlement
```

## Usage

```typescript
import { /* exports */ } from '@noderr/on-chain-settlement';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
