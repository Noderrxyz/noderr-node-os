# @noderr/attestation-mvs

Basic TPM attestation service for Noderr Protocol (signature + nonce + timestamp)

## Installation

```bash
pnpm add @noderr/attestation-mvs
```

## Usage

```typescript
import { /* exports */ } from '@noderr/attestation-mvs';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
