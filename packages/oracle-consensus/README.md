# @noderr/oracle-consensus

Oracle consensus mechanism with BFT algorithm for Noderr Node OS

## Installation

```bash
pnpm add @noderr/oracle-consensus
```

## Usage

```typescript
import { /* exports */ } from '@noderr/oracle-consensus';
```

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
