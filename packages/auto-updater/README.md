# @noderr/auto-updater

Automated version update agent for Noderr Node OS

## Installation

```bash
pnpm add @noderr/auto-updater
```

## Usage

```typescript
import { /* exports */ } from '@noderr/auto-updater';
```

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
