# @noderr/ml-client

gRPC client for Noderr PyTorch ML Service

## Installation

```bash
pnpm add @noderr/ml-client
```

## Usage

```typescript
import { /* exports */ } from '@noderr/ml-client';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
