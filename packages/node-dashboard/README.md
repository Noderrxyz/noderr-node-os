# @noderr/node-dashboard

Real-time monitoring dashboard for Noderr node operators

## Installation

```bash
pnpm add @noderr/node-dashboard
```

## Usage

```typescript
import { /* exports */ } from '@noderr/node-dashboard';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
