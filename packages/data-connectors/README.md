# @noderr/data-connectors

External data connectivity service with free data sources for testnet simulation

## Installation

```bash
pnpm add @noderr/data-connectors
```

## Usage

```typescript
import { /* exports */ } from '@noderr/data-connectors';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
