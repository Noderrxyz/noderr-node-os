# @noderr/market-data

Real-time market data service with historical data replay for testnet simulation

## Installation

```bash
pnpm add @noderr/market-data
```

## Usage

```typescript
import { /* exports */ } from '@noderr/market-data';
```

## API

See [TypeScript definitions](./src/index.ts) for detailed API documentation.

## Development

```bash
# Build
pnpm build

# Test
pnpm test

# Watch mode
pnpm dev
```
